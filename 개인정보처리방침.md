# TodayStep 개인정보 처리방침(초안)

시행일: 2026-02-14  
서비스명: TodayStep

TodayStep(이하 "서비스")는 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다.  
본 방침은 서비스에서 어떤 정보를 처리하는지, 어떻게 보호하는지를 안내합니다.

## 1. 처리하는 정보
서비스는 기본적으로 회원가입 없이 동작하며, 다음 정보를 기기 내 로컬 저장소에 저장할 수 있습니다.

- 할 일 카드 제목/메모/상태/작은일 정보
- 사용 중인 화면 탭 정보
- 일일 자동 등록 동기화 날짜 정보

서비스는 위 정보를 서비스 제공자의 서버로 전송하지 않습니다.

## 2. 정보 처리 목적
- 이용자가 작성한 할 일 데이터를 저장하고 앱 재실행 시 복원하기 위해
- 현재 선택 화면, 진행 상태, 통계 정보를 표시하기 위해
- 일일 자동 등록 기능을 동작시키기 위해

## 3. 처리 및 보관 방식
- 정보는 이용자 기기 내 브라우저(WebView) 로컬 저장소(localStorage)에 저장됩니다.
- 별도 회원 DB 또는 외부 서버에 저장하지 않습니다.
- 보관 기간은 이용자가 앱 데이터를 삭제하거나 앱을 제거할 때까지입니다.

## 4. 제3자 제공 및 처리위탁
- 서비스는 개인정보를 제3자에게 제공하지 않습니다.
- 서비스 운영에 필요한 범위를 제외하고 별도의 개인정보 처리위탁을 하지 않습니다.

## 5. 이용자의 권리 및 행사 방법
이용자는 다음 방법으로 저장 정보를 직접 삭제할 수 있습니다.

- 앱 내 카드 삭제 기능 사용
- 기기 설정에서 앱 데이터 삭제
- 앱 삭제 후 재설치

## 6. 안전성 확보 조치
- 최소한의 정보만 기기 내에 저장합니다.
- 서버 전송을 기본적으로 하지 않아 외부 유출 가능성을 줄입니다.

## 7. 아동의 개인정보
서비스는 만 14세 미만 아동을 대상으로 한 별도 개인정보 수집을 의도하지 않습니다.
